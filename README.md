# MERN CRUD Backend

This is the backend for a MERN stack CRUD application. It is built using Node.js, Express.js, and MongoDB.

## Features


- CRUD operations (Create, Read, Update, Delete)
- RESTful API with Express.js
- MongoDB as the database
